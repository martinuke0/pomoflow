# PomoFlow - Pomodoro Timer + Todo List

A beautiful, modern Pomodoro timer integrated with a todo list built with Next.js 16, TypeScript, and Tailwind CSS.

## Features

- **Pomodoro Timer**: 25min focus / 5min short break / 15min long break with auto-progression
- **Visual Timer**: Animated SVG progress circle
- **Todo Integration**: Add, complete, delete tasks with local persistence
- **Dark Mode First**: Sleek, modern UI optimized for focus
- **Progress Tracking**: Daily pomodoros and cycles count
- **Notifications**: Browser notifications on timer complete
- **Fully Local**: Everything saved in browser localStorage

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React Hooks for state & timer
- LocalStorage persistence
